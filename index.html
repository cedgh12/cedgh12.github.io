4. Détection d'absence par géofencing

MyVoltalis · 2023

Permettre aux utilisateurs d'économiser du chauffage automatiquement, sans y penser. Chaque membre du foyer enregistre son téléphone. Le mode "absence courte" se déclenche dès que tous les téléphones sont hors zone, et s'annule dès qu'un revient.

5. Parcours de saisie du contrat d'électricité

MyVoltalis · 2023 · Intégration API Selectra + Enedis

Pour personnaliser le bilan de consommation, MyVoltalis avait besoin du contrat électricité de chaque adhérent. Parcours multi-étapes : fournisseur, offre, puissance, option tarifaire (Base, HP/HC, Tempo), plages horaires. API Selectra pour les tarifs à jour, API Enedis pour pré-remplir l'option tarifaire et les plages HP/HC.

Discovery, conception du parcours UX complet (cas nominal + cas limites : changement de contrat, déclaration rétroactive), specs, coordination design + engineering, tests, MEP.

75% des adhérents ont déclaré leur contrat · Données fondation du bilan mensuel personnalisé
